# awesome-stars

A curated site of GitHub repos from your GitHub stars, built with TypeScript and HUGO.

## Project Goals
1. Fetch all GitHub stars from authenticated user
2. Classify and organize starred repositories
3. Generate a static site with HUGO showcasing the curated collection
4. Create searchable, categorized directory of quality projects

## Milestones

### Milestone 1: MVP
Build the core pipeline to fetch, classify, and publish starred repos.

- **Phase 1:** Project initialization - TypeScript setup, HUGO scaffolding, GitHub authentication
- **Phase 2:** Data pipeline - Fetch stars, classify by category/topic, store as structured data
- **Phase 3:** Content generation - Convert classified repos into HUGO content, build site

## Success Criteria
- [ ] TypeScript project builds without errors
- [ ] Can authenticate with GitHub API
- [ ] Successfully fetch and classify at least 100 starred repos
- [ ] Generate static HUGO site with all classified repos
- [ ] Site is searchable and browsable by category
